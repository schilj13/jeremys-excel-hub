## Domain Setup (Namecheap → Vercel)

1. Vercel → Project → Settings → Domains → add `jeremyexcel.com` and `www.jeremyexcel.com`
2. Namecheap → Domain List → Manage → Advanced DNS → add:
   - `A Record` | `@` | `76.76.21.21`
   - `CNAME Record` | `www` | `cname.vercel-dns.com`
3. Set TTL to Automatic, save
4. Wait 10–30 min for DNS propagation — Vercel auto-provisions SSL
