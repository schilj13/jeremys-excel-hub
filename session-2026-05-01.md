## How the Kit Form Works

1. User fills out form on `/lp/hr` and clicks submit
2. Browser POSTs to `/api/subscribe` (our Next.js API route)
3. API route calls `https://api.convertkit.com/v3/forms/9391644/subscribe` with the public API key
4. Kit adds subscriber → returns 200
5. Form shows success message ("You're in! Check your inbox…")
6. API key never exposed to browser — stays server-side only
